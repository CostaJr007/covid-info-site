# 🦠 COVID-19 Info Site

Informational website about COVID-19 with resources, statistics, and guidelines.

## 🎯 Features

- ✅ Latest statistics
- ✅ Prevention guidelines
- ✅ Vaccine information
- ✅ Testing locations
- ✅ News feed
- ✅ Responsive design

## 📋 Requirements

- Web browser
- Modern CSS/JS support

## 🔧 Installation

### Local Development

 + "`" + @"
bash
git clone https://github.com/CostaJr007/sitecovid.git
cd sitecovid
 + "`" + @"

Open index.html in your browser.

### Production

Deploy to any static hosting:
- GitHub Pages
- Netlify
- Vercel

## 💡 Usage

Customize content in:
- index.html - Main content
- css/style.css - Styling
- js/main.js - Interactive features
